Uncertainty-Aware Navigation Learning for Robotic Active Visual Search with Multimodal Probabilistic Imitation

Yuhao Wang, Guohui Tian



Abstract

We present a novel framework for visual navigation that integrates Feature-Uncertainty Visual Perception (FUVP) with Cost-Uncertainty Imitation Learning (CUIL). FUVP fuses multimodal visual information-RGB features, semantic segmentation, and object detection-into a ground-projected grid map while explicitly modeling feature uncertainty through Bayesian methods. This approach creates a spatially consistent allocentric representation enhanced by temporal transformers that capture complex dependencies across sequential frames. The system incorporates a rotational sampling mechanism and cross-correlation technique to facilitate robust pose estimation in unknown environments. CUIL complements this perception system with a probabilistic action modeling architecture that quantifies decision uncertainty by representing action costs as Gaussian distributions. Our policy network incorporates an auxiliary inflection point prediction task that identifies topologically significant locations, enabling dynamic loss weighting that prioritizes learning at critical decision points. The combined framework addresses uncertainty propagation throughout the perception-action pipeline. Experimental results demonstrate that this integrated approach significantly improves navigation performance in complex, unexplored environments by maintaining appropriate uncertainty estimates, enhancing both environmental representation and decision-making reliability without reliance on external positioning systems.
